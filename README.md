# ElectoLearn AI 🗳️🤖

**Empowering the next generation of voters with AI-powered interactive education.**

ElectoLearn AI is a production-ready educational platform designed to bridge the knowledge gap for first-time voters and students. Built with a modern tech stack and integrated with Google Gemini, it turns the complex process of democratic elections into an engaging, interactive journey.

## 🌟 Key Features

- **Interactive Election Timeline**: A visual, step-by-step guide through the entire election lifecycle—from voter registration to results.
- **AI Election Assistant**: Powered by Gemini 1.5 Flash, providing neutral, easy-to-understand answers to complex election queries (including a "Explain like I'm 10" mode).
- **Voter Simulator**: An immersive "choose-your-own-adventure" game where users navigate real-life scenarios as either a Voter or a Candidate.
- **Knowledge Quiz**: Challenge your understanding with bilingual (English/Hindi) quizzes and track your progress.
- **Visual Dashboard**: Data-driven insights into historical voter turnout and demographic trends using interactive charts.
- **Bilingual Support**: Full toggle support for English and Hindi to ensure accessibility for a wider audience.

## 🚀 Tech Stack

- **Frontend**: React 19, Vite, Tailwind CSS v4
- **Animation**: Framer Motion
- **AI**: Google Gemini 1.5 Flash API
- **Charts**: Recharts
- **Icons**: Lucide React
- **Routing**: React Router v7

## 🛠️ Setup & Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/SAFAL-TIWARI/ElectoLearnAI.git
   cd ElectoLearnAI
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure Environment Variables**:
   Create a `.env` file in the root directory and add your Gemini API key:
   ```env
   VITE_GEMINI_API_KEY=your_api_key_here
   ```

4. **Run Development Server**:
   ```bash
   npm run dev
   ```

5. **Build for Production**:
   ```bash
   npm run build
   ```
